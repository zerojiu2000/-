#### 餐厅订单数据分析

## 项目简介
基于真实餐厅订单数据的完整数据分析项目，使用Python进行数据处理、统计分析和可视化展示。

## 核心分析
- 📊 菜品销售排行榜Top10
- 💰 订单消费金额分析（总额/平均）
- ⏰ 时间维度分析（小时/日/星期）
- 📈 客户消费行为分析

## 技术栈
- Python 3
- Pandas (数据处理)
- NumPy (数值计算)
- Matplotlib (数据可视化)
- Jupyter Notebook

## 文件说明
- `01餐厅数据分析.ipynb` - 完整分析代码
- `meal_order_detail.xlsx` - 原始数据文件

## 快速开始
1. 安装依赖：`pip install pandas numpy matplotlib jupyter`
2. 运行：`jupyter notebook 01餐厅数据分析.ipynb`
3. 按顺序执行所有代码单元格

## 项目亮点
- 完整的数据分析流程：从数据加载到可视化
- 多维度业务洞察：菜品、时间、订单多角度分析
- 清晰的代码结构和注释
